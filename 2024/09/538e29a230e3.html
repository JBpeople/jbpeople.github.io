1. 基础知识

1.1. 函数参数的常用技巧

1.1.1. 别将可变对象作为参数默认值

Python 函数的参数默认值只会在函数定义阶段被创建一次，之后不论再调用多少次，函数内拿到的默认值都是同一个对象。

1.1.3. 定义仅限关键字参数

建议函数接收参数不要过多，最多不要超过 3 个。如果超过 3 个，函数调用尽量使用关键字参数，这样可以提高代码阅读性。在 Python 中关键字参数如果不做限制，调用是可有可无的。

如果在参数前面加上 * 表示必须使用关键字参数，否则报错。

1.2. 函数返回值的常见模式

1.2.1. 尽量只返回一种数据类型

1.2.2. 谨慎返回 None 值

返回 None 值只有以下 3 种情况：

  1. 操作类函数的默认返回值，例如列表的 append；
  2. 意料之中的缺失值，例如从数据库查找用户，在目录中找一个文件。执行后可能有结果也可能没结果，None 是意料之中的事情。

1.2.3. 早返回，多返回

“单一出口”出现于汇编和 FORTRAN 语言时代，编程语言可以指定代码跳转位置。现在 Python 没有必要坚持“单一出口”，早返回反而是给阅读人减负。
