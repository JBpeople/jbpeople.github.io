发表于2024-04-04|电脑Mac
按照下面所示命令执行，编辑zshrc文件，在文件中添加以下内容。 123456789101112131415vim ~/.zshrc# where proxyproxy () { export http_proxy="http://127.0.0.1:7890" export https_proxy="http://127.0.0.1:7890" echo "HTTP Proxy on"}# where noproxynoproxy () { unset http_proxy unset https_proxy echo "HTTP Proxy off"} 注意替换自己的端口号，个人使用的是clash所以默认端口为7890。以后打开终端输入proxy即开启代理，输入noproxy即关闭代理。
